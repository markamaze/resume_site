NoteStacks

Summary

A concept I came up with for taking notes for what I am working on, is to use post-it notes the same way as the Stack data-structure. This project was built as a way to do that same note-taking practice in an app, while learning how to work with the Android framework.

User Details

This simple note taking app, was designed with the intention of simulating a note-taking process where the first note of the stack is your starting point or overview of what you are working on. From that, you choose what to start with, and this can be represented by putting a note on the top of the stack and maybe breaking that down into smaller tasks. By stacking notes in this manner, you can easily keep track of complicated projects. The top of the stack is always where you left off. The previous note is what led you to that task. When you've finished everything on the top note, just delete it and the note on top will be where you branched off from. Once you're stack is back to the first note, and everything on it is done, that would mean your project is done.

Technical Details

Notestacks was built as an introduction to Android. There is not much significant about the design, just a practice in building something with the android framework. It utilizes two activities and object classes and a touch listener.
